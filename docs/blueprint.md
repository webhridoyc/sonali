# **App Name**: Sonali Cooperative Portal

## Core Features:

- Homepage Hero Slider: Displays high-definition images of cooperative activities with captions.
- Services Grid: Interactive cards linking to detailed information on livestock, textiles, handicrafts, and pottery projects.
- Member Portal: Allows members to download forms (application, audit reports, bylaws) and submit online applications.
- Contact Form: A form to submit inquiries with fields for Name, Phone, and Inquiry, plus links to the Society's social media presence.
- Bilingual Toggle: Allows users to switch between Bangla (primary) and English.

## Style Guidelines:

- Primary color: Deep Forest Green (#1B5E20) to represent growth and agriculture.
- Secondary color: Golden Yellow (#FFB300) to symbolize prosperity.
- Background: Off-white or light gray to enhance readability.
- Support for high-quality Bangla font: 'Hind Siliguri' (or 'Kalpurush') for Bangla text (serif).
- Support for modern English font: 'Poppins' (sans-serif) for a clean, contemporary look.
- Simple, relevant icons to represent each service (livestock, textiles, handicrafts, pottery).
- Mobile-responsive design, ensuring optimal viewing experience on Android phones.
- Sticky header with logo, menu, and 'Call Now' button for easy navigation and contact.
